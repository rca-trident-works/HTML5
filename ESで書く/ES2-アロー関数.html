アロー関数

従来の関数宣言よりも短く書くことができる。色々なところで有用である。
デベロッパーツールで動作を確認しながら読み進めること。

1. 引数なし

いままでの記述方法


function hello(){
    alert("Hello world!");
}
hello();

アロー関数使用


const hello = () =>{
    alert("Hello world!");
}
hello();

省略形


const hello = () => alert("Hello world!");
hello();

※関数は1行の場合、中括弧やreturnを省略できる。

２. 引数あり

いままでの記述方法


const checkAmpm = (currentHour, currentMinute) => {
if (currentHour < 12) {
    return "AM";
} else if (currentHour > 12) {
    return "PM";
} else {
    //三項演算子
    return currentMinute === 0 ? "noon" : "PM";
}};

const result = checkAmpm(20, 30);
console.log(result); // "PM"

アロー関数使用


const checkAmpm = (currentHour, currentMinute) => {
if (currentHour < 12) {
    return "AM";
} else if (currentHour > 12) {
    return "PM";
} else {
    //三項演算子
    return currentMinute === 0 ? "noon" : "PM";
}};

const result = checkAmpm(20, 30);
console.log(result); // "PM"

※引数が一つの時は「currentHour => ～」と()を省略することができる。

練習問題：アロー関数への書き換え

以下の従来の関数宣言をアロー関数に書き換えよ。
※それぞれの関数をアロー関数に書き換え、可能な場合は省略形も示せ。 プログラムは下部のpre,codeタグ内に、動作確認後に記述せよ。 


// 問題1: 引数なしの関数
function greet() {
    return "Hello, World!";
}
    
// 問題2: 引数ありの関数
function multiply(a, b) {
    return a * b;
}
    
// 問題3: 条件分岐を含む関数
function checkAge(age) {
    if (age >= 18) {
        return "成人です";
    } else {
        return "未成年です";
    }
}
    
// 問題4: オブジェクトを返す関数
function createPerson(name, age) {
    return {
        name: name,
        age: age
    };
}
    
// 問題5: 即時実行関数
(function() {
    console.log("This function runs immediately");
})();


// 問題1: 引数なしの関数
const greet = () => "Hello, World!";

// 問題2: 引数ありの関数
const multiply = (a, b) => a * b;

// 問題3: 条件分岐を含む関数
const checkAge = age => age >= 18 ? "成人です" : "未成年です";

// 問題4: オブジェクトを返す関数
const createPerson = (name, age) => ({ name, age });

// 問題5: 即時実行関数
(() => console.log("This function runs immediately"))();
